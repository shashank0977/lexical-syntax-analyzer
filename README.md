# Lexical Syntax Analyzer

A Python-based mini compiler project implementing core compiler design concepts including lexical analysis, grammar handling, FIRST & FOLLOW computation, and LL(1) parsing.

---

## 🚀 Features

- Lexical Analysis (Tokenization)
- Grammar Input & Storage
- FIRST Set Computation
- FOLLOW Set Computation
- LL(1) Parsing
- Menu-driven Command Line Interface

---

## 🛠 Technologies Used

- Python 3
- Regular Expressions (re module)
- Compiler Design Concepts

---

## 📚 Concepts Covered

- Compiler Phases
- Context-Free Grammar (CFG)
- FIRST & FOLLOW Sets
- LL(1) Parsing Technique
- Syntax Analysis
