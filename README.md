# Compiler Design Laboratory

## Project Overview

This project implements core Compiler Design laboratory experiments in a modular and menu-driven Python application. It demonstrates fundamental concepts of lexical analysis, automata theory, and syntax analysis.

## Implemented Experiments

1. Lexical Analyzer  
2. Regular Expression to NFA (Basic Thompson Construction)  
3. NFA to DFA (Subset Construction Demonstration)  
4. Grammar Transformations (Left Recursion Elimination)  
5. FIRST Set Computation  
6. Predictive Parsing Table Construction (LL(1) Demonstration)

## Requirements

- Python 3.x
- No external libraries required

---

## How to Run

1. Open terminal inside the project root folder.
2. Run:

   python main.py

   or

   python3 main.py

3. Select the desired experiment from the menu.
